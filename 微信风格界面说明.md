## 🎨 主要特性

### 三栏布局
1. **左侧导航栏（80px宽）**
   - 用户头像
   - 功能图标（消息、联系人、文件、收藏）
   - 设置和更多菜单

2. **中间对话列表栏（280px宽）**
   - 搜索框
   - 对话列表（按时间排序）
   - 每个对话项显示头像、名称、最新消息、时间

3. **右侧聊天窗口栏（自适应宽度）**
   - 顶部标题栏（显示数字人信息）
   - 消息显示区域（微信风格气泡）
   - 数字人视频窗口（可折叠）
   - 输入区域（支持多行输入）

### 企业级渐变配色
- 主渐变：`linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
- 侧边栏：深色背景 `#2e2e2e`
- 消息气泡：发送 `#95ec69`，接收 `#ffffff`

### 数字人视频窗口
- 可折叠/展开
- 显示远程视频和本地视频
- 加载状态提示

## 🚀 使用方法

1. **启动后端服务**
   ```bash
   python app.py
   ```

2. **访问前端**
   - 打开浏览器访问：`http://127.0.0.1:8282/ui/videochat.html`
   - 或通过主入口：`http://127.0.0.1:8282/`

3. **功能使用**
   - 点击权限覆盖层获取摄像头/麦克风权限
   - 在输入框输入消息，按Enter发送
   - 点击数字人视频窗口的折叠按钮可折叠/展开
   - 使用视频控制按钮切换摄像头/麦克风
