# 前端构建步骤（详细版）

## 为什么需要安装Node.js？

前端代码是用Vue 3 + TypeScript + Vite开发的，需要Node.js来：
1. 安装依赖包（vue-router、ant-design-vue等）
2. 编译TypeScript代码
3. 打包构建成浏览器可以运行的代码

## 安装Node.js

### Windows系统

1. **访问官网下载**
   - 打开浏览器访问：https://nodejs.org/
   - 点击下载"LTS"版本（长期支持版，更稳定）
   - 下载完成后运行安装程序

2. **安装过程**
   - 一路点击"下一步"即可
   - 安装程序会自动添加到系统PATH
   - 安装完成后**重启命令行窗口**（重要！）

3. **验证安装**
   打开新的命令行窗口（PowerShell或CMD），输入：
   ```bash
   node --version
   npm --version
   ```
   如果显示版本号，说明安装成功！

## 构建前端

### 步骤1：进入前端目录
```bash
cd src\handlers\client\rtc_client\frontend
```

### 步骤2：安装依赖
```bash
npm install
```
这会安装所有需要的包，可能需要几分钟时间。

### 步骤3：构建前端
```bash
npm run build
```
这会编译和打包前端代码，生成新的`dist`目录。

### 步骤4：重启后端服务
构建完成后，重启`app.py`，然后访问：
- 视频聊天：`http://localhost:8282/ui/videochat`
- 控制台：`http://localhost:8282/ui/console`

## 常见问题

### Q: 安装Node.js后命令还是找不到？
A: 需要**重启命令行窗口**，或者重启电脑。

### Q: npm install很慢？
A: 可以使用国内镜像：
```bash
npm config set registry https://registry.npmmirror.com
```

### Q: 构建失败怎么办？
A: 检查错误信息，通常是：
- 缺少依赖：重新运行`npm install`
- 版本不兼容：确保Node.js版本>=18

## 总结

**必须安装Node.js**，因为：
1. 前端代码已经更新（添加了控制台、路由等）
2. 旧的`dist`目录不包含新功能
3. 必须重新构建才能看到新的界面

安装Node.js是一次性的，之后就可以随时构建前端了！
