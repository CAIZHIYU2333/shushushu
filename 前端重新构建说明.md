# 前端重新构建说明

## 问题原因
前端代码已经更新，但是需要重新构建才能生效。前端构建后的文件在 `dist` 目录中，浏览器加载的是构建后的文件，而不是源代码。

## 解决步骤

### 1. 进入前端目录
```bash
cd src/handlers/client/rtc_client/frontend
```

### 2. 安装依赖（包括新添加的 vue-router）
```bash
pnpm install
```
或者如果使用 npm：
```bash
npm install
```

### 3. 重新构建前端
```bash
pnpm run build
```
或者如果使用 npm：
```bash
npm run build
```

### 4. 重启后端服务
构建完成后，需要重启 `app.py` 服务，新的前端代码才会生效。

## 新增功能

1. **控制台入口**：在视频聊天页面右上角添加了一个设置按钮，点击可以进入控制台
2. **控制台功能**：
   - 访问 `/console` 或点击右上角设置按钮进入控制台
   - 包含权限管理、数字人管理、知识库、数据可视化、人物模板等功能
3. **UI改进**：视频聊天界面样式已优化，包含渐变背景和动画效果

## 访问控制台
- 直接访问：`http://localhost:8282/console`（根据你的端口调整）
- 从视频聊天页面：点击右上角的设置按钮
