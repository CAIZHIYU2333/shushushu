# 前端文件说明

## ✅ 新的原生HTML/CSS/JS文件位置

所有新的前端文件都在以下目录：
```
src/handlers/client/rtc_client/frontend/
```

## 📁 主要文件列表

### 入口文件
- `index.html` - 主入口页面（路由管理）
- `app.js` - 路由系统
- `styles.css` - 全局样式

### 视频聊天页面
- `videochat.html` - 视频聊天页面结构
- `videochat.css` - 视频聊天样式
- `videochat.js` - 视频聊天逻辑

### 控制台页面
- `console.html` - 控制台主页面
- `console.css` - 控制台样式
- `console.js` - 控制台逻辑

### 控制台子模块
- `console-dashboard.html/css/js` - 仪表盘
- `console-permissions.html/css/js` - 权限管理
- `console-avatars.html/css/js` - 数字人管理
- `console-knowledge.html/css/js` - 知识库
- `console-analytics.html/css/js` - 数据分析
- `console-templates.html/css/js` - 模板管理

### 工具类文件
- `api.js` - API请求封装
- `state.js` - 状态管理
- `webrtc.js` - WebRTC工具
- `ws.js` - WebSocket封装
- `stream-utils.js` - 媒体流工具
- `player.js` - 音频播放器
- `gaussian-avatar.js` - 高斯数字人
- `binary-utils.js` - 二进制工具
- `utils.js` - 通用工具

## 🔍 如何验证是否使用了新文件

1. **查看后端启动日志**：
   启动后端时，应该看到：
   ```
   ============================================================
   ✅ 使用新的原生HTML/CSS/JS前端（非Vue版本）
   前端路径: .../frontend
   ============================================================
   ```

2. **查看浏览器控制台**：
   - 打开浏览器开发者工具（F12）
   - 查看Network标签
   - 刷新页面
   - 应该看到加载的是 `index.html`, `app.js`, `styles.css` 等文件
   - **不应该**看到 `assets/index.js`, `assets/index.css` 等Vue构建文件

3. **查看页面源码**：
   - 右键页面 -> 查看页面源码
   - 应该看到简单的HTML结构，包含 `<script src="app.js">`
   - **不应该**看到复杂的Vite构建脚本

## 🚨 如果还是看到Vue版本

1. **清除浏览器缓存**：
   - 按 `Ctrl + Shift + Delete`
   - 清除缓存和Cookie
   - 或者使用无痕模式（Ctrl + Shift + N）

2. **检查后端日志**：
   - 确认后端启动时显示的是"使用新的原生HTML/CSS/JS前端"
   - 如果显示"使用旧的Vue构建版本"，说明文件检测有问题

3. **重启后端服务**：
   - 停止当前后端
   - 重新启动后端
   - 查看启动日志

4. **直接访问文件**：
   - 访问 `http://127.0.0.1:8282/ui/index.html`
   - 应该看到简单的加载页面

## 📝 文件结构对比

### 新版本（原生HTML/CSS/JS）
```
frontend/
├── index.html          ← 简单HTML，直接引用app.js
├── app.js              ← 原生JavaScript路由
├── styles.css          ← 原生CSS
├── videochat.html      ← 独立HTML文件
├── videochat.js        ← 原生JavaScript
└── ...
```

### 旧版本（Vue构建）
```
frontend/dist/
├── index.html          ← 包含Vite构建脚本
└── assets/
    ├── index.js        ← 打包后的Vue代码
    ├── index.css       ← 打包后的CSS
    └── ...
```
